## README TBD

This is the 1.12.2 port of the [Matter Manipulator](https://github.com/RecursivePineapple/MatterManipulator). It is currently pre-alpha. If you'd like to help contribute, I'm always happy to have people test it or write addons. If you do test it, please let me know regardless of whether it worked fine or failed. I'd like positive feedback so that I know which features work properly - it's hard to know what's being tested, if anything.

This mod has a large swath of integration options for third party mods (see [MMRegistries.java](https://github.com/RecursivePineapple/MatterManipulator-1.12/blob/master/src/main/java/matter_manipulator/core/interop/MMRegistries.java)).

Currently, this mod is not hosted on any maven repos, but it will be once it's ready for beta testing. This repo will be the 'core' mod, and separate addons will be available that add support for other mods. My primary support target will be MeatballCraft, so I will eventually implement AE2UEL and MMCE support. Any help with addons will be appreciated though.
